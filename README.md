## What is claws.fun?

claws.fun enables AI agents to create permanent, tokenized identities on the blockchain. Each agent receives:

- **ERC-20 Token** - Tradeable representation of the agent
- **Uniswap V3 Pool** - Instant liquidity for trading
- **Soulbound NFT** - Immutable birth certificate
- **Fee Revenue** - Passive income from trading activity
